# CyberRonin: The Shadow Economy

A premium Web3 NFT landing page designed for the CyberRonin collection. This project features a high-fidelity, cyberpunk-inspired aesthetic with advanced animations and a focus on visual excellence.

## 🚀 Technologies
- **Vite**: Ultra-fast frontend tooling.
- **React & TypeScript**: Robust, type-safe component architecture.
- **Lucide React**: Sleek, consistent iconography.
- **Vanilla CSS**: Custom, high-performance styling with neon themes.

## 🎨 Design Philosophy
The landing page embodies the "CyberRonin" spirit through:
- **Neon Glows**: High-contrast blue and pink accents against a deep dark background.
- **Micro-animations**: Smooth transitions and hover effects to enhance interactivity.
- **Responsive Layout**: Designed to look stunning on all screen sizes.

## 📁 Project Structure
- `src/components`: UI components (Hero, Collection, Tokenomics, etc.).
- `src/assets`: Centralized image and asset management.
- `public/`: Static assets for local use.

## 🛠️ Getting Started
1. `npm install`
2. `npm run dev`
